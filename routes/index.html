Dev-Resources API Documentation

This will server as basic documentation of the API and detail the various endpoints available

User Endpoints

  • /user/auth

    A get request to authenticate users

  • /user/new

    Post req to register a new user, not require if we use discord auth

Resource Endpoints

  • /resource/all

    A get request to list all resources

  • /resource/:user/bookmark

    A get req to return all resource the specified user has bookmarked

  • /resource/:user/bookmark

    A post req to save a resource for the specified user
